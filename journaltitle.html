Looking for an Article?

Search Tip A:
 

This search retrieves printed and electronic journals, magazines, newspapers, and other serials listed in the Library Catalog. Not all electronic journals are listed in the Catalog, so if you don't find the title you're looking for try "Search by Title for an Electronic Journal, Magazine or Newspaper" below.

Type as much or as little of the journal title as you want. Do not search title abbreviations (such as J Biol Chem for Journal of Biological Chemistry) here.

For Example:

  • journal of cell biology
  • journal of cel
Search Tip B:
 

This database contains only journals, magazines, newspapers, and other serials for which some electronic content is available. If you don't find the title you're looking for here, or if the volume/year you need is not available electronically, try "Search by Title for a Journal, Magazine or Newspaper" above to find it in print.

Type as much or as little of the exact journal title as you wish, or type words from the title. Some abbreviated titles are included. Select "Contains" for title keyword search. Select "Starts with" to search for the exact title.

For Example:

  • Starts with: harvard bus
  • Contains: journal biological
